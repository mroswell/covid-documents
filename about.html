About the EUA File Navigator

Overview

This navigator provides access to clinical trial documents submitted to the FDA for the Pfizer-BioNTech COVID-19 vaccine Emergency Use Authorization (EUA). These documents were obtained through Freedom of Information Act (FOIA) requests and represent a comprehensive collection of clinical trial data, safety reports, and efficacy analyses.

Document Collection

The collection includes 854 documents spanning multiple modules of the regulatory submission:

  • Module 1 (M1): Administrative and prescribing information
  • Module 2 (M2): Summaries and overviews
  • Module 4 (M4): Nonclinical study reports
  • Module 5 (M5): Clinical study reports

Data Standards

The clinical trial data follows CDISC (Clinical Data Interchange Standards Consortium) standards:

  • SDTM (Study Data Tabulation Model): Standardized format for organizing clinical trial data
  • ADaM (Analysis Data Model): Datasets prepared for statistical analysis
  • Supplemental Qualifiers: Additional data that extends the standard domains

Using the Navigator

The file navigator allows you to explore documents by various criteria:

  • Filter by file type (PDF, Word, Excel, etc.)
  • Filter by domain (adverse events, laboratory data, demographics, etc.)
  • Filter by folder (different submission dates)
  • Filter by module (M1, M2, M4, M5)
  • Sort by document count or alphabetically by tag name
Note: This is an unofficial resource created to help researchers and the public navigate the FOIA'd documents. Always refer to official FDA sources for authoritative information.
