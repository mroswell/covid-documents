These Define-XML files contain comprehensive metadata documentation for the COVID-19 vaccine clinical trial datasets submitted to FDA. Each file provides detailed information about dataset structures, variables, controlled terminology, and computational methods used in the trials.

About Define-XML Files

ADaM (Analysis Data Model): Contains analysis-ready datasets used for statistical analysis and regulatory submissions. These datasets are derived from SDTM data and include additional variables needed for analysis.

SDTM (Study Data Tabulation Model): Contains raw clinical trial data in a standardized format. These datasets capture the original observations and measurements from the clinical trial.

CDISC Standards: Both ADaM and SDTM follow CDISC (Clinical Data Interchange Standards Consortium) standards to ensure consistency and interoperability in clinical research data.

Data Sources: These files document clinical trial data from both Moderna's mRNA-1273 vaccine (FDA-CBER-2022-1614) and Pfizer-BioNTech's BNT162b2 vaccine trials submitted to FDA.
